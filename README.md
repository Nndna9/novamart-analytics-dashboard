# Marketing Analytics Dashboard

## Overview
The Marketing Analytics Dashboard is an interactive data visualization tool designed for NovaMart, an omnichannel retail company. This dashboard allows users to explore marketing performance, customer behavior, product sales, and the effectiveness of lead scoring models through various visualizations.

## Project Structure
The project consists of the following files:
- `app.py`: The main application file that contains the code for loading data, creating visualizations, and setting up the dashboard's interactive components.
- `requirements.txt`: A file that lists all the necessary Python libraries required to run the Streamlit application.
- `README.md`: This documentation file that provides an overview of the project and instructions for setup and usage.

## Setup Instructions
1. **Clone the Repository**
   Clone this repository to your local machine using:
   ```
   git clone <repository-url>
   ```

2. **Navigate to the Project Directory**
   Change your working directory to the project folder:
   ```
   cd marketing-analytics-dashboard
   ```

3. **Install Required Libraries**
   Use the following command to install the required libraries listed in `requirements.txt`:
   ```
   pip install -r requirements.txt
   ```

4. **Run the Streamlit Application**
   Start the Streamlit application by running:
   ```
   streamlit run app.py
   ```

5. **Access the Dashboard**
   Open your web browser and go to `http://localhost:8501` to view the dashboard.

## Features
- Interactive visualizations for marketing performance metrics.
- Insights into customer behavior and product sales.
- Evaluation of lead scoring model performance.
